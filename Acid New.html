Links got you feeling blue?

by Senior Email Developer Megan Boshuyzen

You’ve coded your email, and it’s obviously gorgeous with a nice bold, brand color for your text links.

Except when you test your email using Campaign Precheck … they’re blue! Ugh, annoying!

Luckily, banishing blue links is pretty simple. Add the following CSS snippets to your base code, and never see a blue text link again.

Apple 

a[x-apple-data-detectors] {
        color: inherit !important;
        text-decoration: none !important;
        font-size: inherit !important;
        font-family: inherit !important;
        font-weight: inherit !important;
        line-height: inherit !important;
}

Gmail 

u+#body a {
        color: inherit !important;
        text-decoration: none !important;
        font-size: inherit !important;
        font-family: inherit !important;
        font-weight: inherit !important;
        line-height: inherit !important;
}

Samsung 

#MessageViewBody a {
         color: inherit !important;
         text-decoration: none !important;
         font-size: inherit !important;
         font-family: inherit !important;
         font-weight: inherit !important;
         line-height: inherit !important;
}
